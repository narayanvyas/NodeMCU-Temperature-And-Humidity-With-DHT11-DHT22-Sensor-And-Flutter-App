# NodeMCU ESP8266 Temperature And Humidity With DHT11 / DHT22 Sensor And Flutter App
NodeMCU ESP8266 Temperature And Humidity With DHT11 / DHT22 Sensor And Flutter App.

## Prerequisites
1. NodeMCU ESP8266
2. 10KOhm Resistor
3. DHT11 / DHT22
4. jumper wires  
5. Breadboard
